# 视觉重构说明（v0.5）

> 依据 [taste-skill](https://github.com/leonxlnx/taste-skill) 约束彻底重做 UI / 动效 / 布局。
> 基线：`DESIGN_VARIANCE 8 · MOTION_INTENSITY 6 · VISUAL_DENSITY 4`。

## 一、清掉的「AI 味」雷区

| AI 雷区 | 旧版 | 重构后 |
|---|---|---|
| **Emoji 图标** | 🌊 logo、◈▤✦✎☺ 导航、🔒▶⏸⏱▲🔴 | 全部替换为 **Phosphor 线性图标** + 自绘 TideMark（31 处 → 0） |
| **居中 Hero** | 居中大标题 | **非对称分栏**：左文案 / 右产品掠影（mesh 渐变 + 悬浮 chips） |
| **三等分卡片** | 3 个等宽卡片 feature row | **Bento**（1 大 + 2 小）+ **zig-zag** 赛道行（左右交替） |
| **AI 紫/蓝 + 双强调色** | teal + dawn 橙双色 | **单强调色**：深潮汐青 `#1f6b60`（饱和度 <80%），去橙 |
| **系统字体 / Inter** | PingFang 系统栈 | **Geist Sans**（拉丁）+ **Geist Mono**（所有数字/overline）+ CN 系统栈 |
| **纯黑 / 硬阴影** | `#000`、通用 box-shadow | 近黑 `#17181a`；阴影**染背景色**（`rgba(13,51,45,…)`） |
| **卡片堆叠** | 处处 `border + card` | 榜单/数据条改用 **border-t / divide-y / 负空间** |

## 二、动效系统（MOTION_INTENSITY 6）

全部隔离在 `src/components/motion.tsx`（Client Component，仅动 `transform/opacity`）：

- **Reveal / Stagger** — 滚动进入的上浮 + 瀑布式交错揭示（`whileInView`, spring ease）
- **Magnetic** — 主 CTA 向光标磁吸（`useMotionValue`，不触发 re-render）
- **Spotlight** — 课程卡指针跟随高光边框
- **CountUp** — 数据条数字滚动
- CSS 层：`--ease-out-expo` 缓动、skeleton **微光扫过**、`live-dot` **呼吸**状态点、赛道**无限跑马灯**、链接下划线生长、`:active` 触感下压（`scale-[0.985]`）
- 全量 `prefers-reduced-motion` 降级

## 三、设计 Token（`globals.css`）

- 单强调色 `accent-{50..900}`（深潮汐青）
- 中性基座 `ink-{50..950}`（暖 stone，无纯黑）+ `paper` / `paper-raised`
- `--font-mono` 用于数字、`.overline` 标签
- `--radius-card: 20px`、`--shadow-soft`（染色）、mesh 有机渐变背景
- 细颗粒噪点固定层（`body::before`，`pointer-events-none`，不随滚动重绘）

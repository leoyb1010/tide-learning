## P0 · 必修（用户可感知的错误，~半小时一个 PR）

### P0-1 密码提示矛盾（注册被误导）
- **位置**：`src/app/login/page.tsx:69`（placeholder「至少 6 位」）vs `src/lib/session.ts:33`（后端强度 ≥8 位 + 字母 + 数字）
- **现象**：用户按提示输 6 位密码注册 → 提交才报「密码至少 8 位」
- **修法**：placeholder 改「至少 8 位，含字母和数字」；`SettingsSections.tsx:77` 已是正确文案，三处对齐
- **工作量**：5 分钟

### P0-2 iOS/Mac 登录文案未同步用户名登录
- **位置**：`ios/YoudaoStudio/Features/Auth/LoginView.swift:25`、`ios/YoudaoStudio/MacFeatures/MacLoginView.swift:46` 写死「手机号 / 邮箱」
- **现象**：后端已支持 `dingyue`/`admin` 用户名登录，原生端用户不知道；体验账号提示若写旧邮箱也要同步
- **修法**：文案改「用户名 / 手机号 / 邮箱」+ 体验账号提示同步 `dingyue/demo123 · admin/admin123`；跑 iOS+Mac build 验证
- **工作量**：15 分钟 + 双端 build

## P1 · 本周建议（体验/性能/健壮性）

### P1-1 首页大图 LCP 优化
- **位置**：`src/components/home/ActOne.tsx`（教室背景两张 ~273KB）、`ActTwo.tsx`（书脊图 189KB）
- **修法**：首屏教室图加 `fetchPriority="high"`（或 `<link rel=preload>`），第二幕书脊图确认 `loading="lazy"`（在视口外才对）；补 `sizes`
- **工作量**：20 分钟 + Lighthouse 对比

### P1-2 课程抽屉移动端触屏打磨
- **位置**：`src/components/home/HomeFunnel.tsx:346,379`
- **问题**：① snap 快滑可能停在卡中间；② 触屏长按后 `:hover` 放大态残留
- **修法**：`.drawer-card` 补 `scroll-snap-stop` 调优；`@media (hover: none)` 下关闭 hover 浮升与 `will-change`
- **工作量**：30 分钟 + 真机/模拟器验证

### P1-3 iOS 深链 URL scheme 缺失（记忆中的已知 backlog，审计实锤）
- **位置**：`ios/project.yml` iOS target 无 `CFBundleURLTypes`（Mac 的 Info-Mac.plist:21-31 已配 `youdaostudio://`）
- **现象**：深链在 iOS 上无法唤起 App
- **修法**：project.yml iOS target 补 URL scheme + xcodegen 重生成 + iOS build + 模拟器 `xcrun simctl openurl` 实测
- **工作量**：30 分钟

### P1-4 契约测试补 username 登录覆盖
- **位置**：`tests/contract.test.ts`（现只测 email 登录）
- **修法**：补 `dingyue`（username）与 `13900000000`（phone）两条登录用例，防止未来重构悄悄破坏三路登录
- **工作量**：15 分钟

### P1-5 LLM prompt 拼接加固
- **位置**：`src/app/api/ai/generate-course/route.ts:67`、`import-source/route.ts:79-80`（用户输入直接模板拼接）
- **风险**：特殊字符破坏输出 JSON 结构 / prompt 注入引导跑偏（LLM 应用无法根除，但可降低）
- **修法**：用户输入以 `JSON.stringify` 包裹或明确定界符包围 + 输出侧已有规范化则补齐失败重试；`generate-course` 补 prompt 最短长度校验（如 ≥4 字）
- **工作量**：40 分钟 + AI 生成回归（需 DeepSeek 余额）

## 二、logo 资源（真实矢量提取）

- `public/brand/youdao-ink.png` — 近黑，用于正文/页脚

## 三、融入位置

- **顶部导航**：`有道 youdao │ 潮汐学习` 联合标识（母品牌背书 + 产品名），复刻 VIS lockup 范式
- **首页 Hero**：`网易有道 出品` 背书 chip（含 mini logo）
- **页脚**：有道 wordmark + `网易有道 出品` 说明 + 版权 `© 2026 网易有道 · 潮汐学习`
- **全站强调色**：teal → **有道红 `#FC011A`**（CTA、激活态、链接、封面渐变、mesh 背景、投票、状态点）

## 四、品牌与 taste-skill 的平衡

taste-skill 默认「去高饱和 / 去大红促销」，但**品牌识别优先**（skill 亦声明 "unless explicitly requested"）。
处理原则——**红色作为有目的的强调，而非满屏铺红**：

- 中性基座（暖白 paper + 近黑 ink）承载结构与留白
- 红色只用于品牌标识、主 CTA、激活态、关键高亮
- 封面/播放器用「深炭 → 品牌红」渐变，克制不刺眼
- 阴影染红色调、选中态红、focus 环红——细节统一
- 不做倒计时逼单、不满屏红卡片（延续产品「安静高级」调性）
